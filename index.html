YouTube Downloader & Transcriber

מכין את הורדת האודיו...

מבצע תמלול... תהליך זה עשוי להימשך מספר דקות

תוצאות התמלול:

הורד JSON הורד SRT הורד טקסט

ממשקי API זמינים:

GET /download?id=YOUTUBE_VIDEO_ID

מוריד את האודיו (MP3) של סרטון YouTube.

GET /transcribe?id=YOUTUBE_VIDEO_ID&format=json|srt|txt

מתמלל סרטון YouTube ומחזיר את התמלול בפורמט המבוקש.

GET /proxy?url=URL

פרוקסי כללי להורדת קבצים.

הערות:

1. הגבלת קצב בתוקף. מקסימום 10 בקשות לכל כתובת IP בכל 60 שניות.

2. תמלול עשוי להימשך מספר דקות, בהתאם לאורך הסרטון.

3. שרת זה נועד למטרות לימודיות בלבד.
